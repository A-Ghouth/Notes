CSS

  1. Selectors:
    • Element Selector: element
    • Class Selector: .classname
    • ID Selector: #elementID
    • Attribute Selector: [attribute="value"]
  1. Box Model:
    • Width and Height: width, height
    • Margin: margin
    • Padding: padding
    • Border: border
  1. Typography:
    • Font Family: font-family
    • Font Size: font-size
    • Font Weight: font-weight
    • Text Color: color
    • Text Alignment: text-align
  1. Layout and Positioning:
    • static: The default position.
    • relative: Allows editing the position in flow with the parent position as (0,0).
    • absolute: Takes the parent position as (0,0) and can be freely adjusted.
    • fixed: Becomes absolute but remains fixed even when scrolling.
    • sticky: A relative position that becomes fixed when certain conditions are met.
  1. Colors and Backgrounds:
    • Background Color: background-color
    • Background Image: background-image
    • Opacity: opacity
    • colo = hsl(color, saturation, lightness)
  1. Responsive Design:
    • Media Queries: @media
    • Responsive Units: em, rem, %
  1. Flexbox:
    • Container: display: flex
    • Justify Content: justify-content
    • Align Items: align-items
  1. Grid:
    • Container: display: grid
    • Grid Template Columns: grid-template-columns
    • Grid Template Rows: grid-template-rows
    • Grid Gap: grid-gap
    • Grid Column: grid-column
    • Grid Row: grid-row
    • Grid Area: grid-area
    • display: grid: Sets the container as a grid container.
    • repeat(auto-fit, 1fr): Creates template areas with automatic fitting.
    • grid-area: Assigns areas to grid items.
    • justify-content: Controls the alignment along the y-axis.
  1. Transitions and Animations:
    • Transition: transition-property, transition-duration, transition-timing-function, transition-delay
    • Animation: @keyframes, animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count
  1. Pseudo-classes and Pseudo-elements:
    • Pseudo-classes: :hover, :focus, :nth-child()
    • Pseudo-elements: ::before, ::after
  1. CSS Units:
    • Pixels (px)
    • Percentages (%)
    • Em (em)
    • Rem (rem)
  1. CSS Variables:
    • Declare: -variable-name
    • Use: var(--variable-name)
  1. Other
    • Transform: transform
    • Filter: filter
    • Background Blend Mode: background-blend-mode

    • Bootstrap
