General

Virtual Environment

  • Create a virtual environment:
    • python3 -m venv myenv
  • Activate the virtual environment:
    • source myenv/bin/activate
  • Deactivate the virtual environment:
    • deactivate
  • Install Django using pipenv:
    • pipenv install

Steps

  1. Install Django.
  1. Create a Django project.
  1. Create a Django app.
  1. Add the app to the project's settings.
  1. Work on views.
  1. Create URL patterns by creating a URLs file inside the app.
    • Define a URL using app_name.
  1. Create templates inside the app.
    • Use a layout.html file.
  1. Create a static folder for CSS files.
    • Load static files using {% load static %}.
    • Use {% static 'newyear/styles.css' %} to specify the static file path.
    • Collect static files using python manage.py collectstatic.
  1. Implement forms.
    • Use {% csrf_token %} inside the form.
    • Define form classes using forms.Form.
    • Use forms.CharField for input fields.
    • Add labels to form fields.
  1. Handle static assets.
  1. ...

Django HTML Language

  • Conditional statements:
    • Example: {% if newyear %} <h1>YES</h1> {% else %} <h1>NO</h1> {% endif %}
  • Simple variable rendering:
    • Example: <h1>Hello, {{ name }}!</h1>
  • Template inheritance using blocks:
    • Example:
      {% extends "tasks/layout.html" %}

{% block body %}
{% endblock %}
  • Looping over items:
    • Example:
      {% for task in tasks %}
    <li>{{ task }}</li>
{% endfor %}
  • Generating URLs:
    • Example:
      {% url 'index' %}

<a href="{% url 'tasks:index' %}">View Tasks</a>
  • Rendering forms:
    • Example:
      {{ form }}
<input type="submit">
  • ...
