HTML

  • document object:
    • Creating elements:
      • createElement("h1") to create an <h1> element.
      • Use methods like append, prepend, and insertBefore to add elements.
    • Modifying elements:
      • .innerHTML to modify the HTML content (not secure for user input).
      • .textContent to set the text content (more secure, supports HTML tags).
    • Calling events:
      • Example: onClick = "invokeFunction()".

CSS

  • .style property to modify CSS attributes.
  • classList object for working with CSS classes:
    • .add, .remove, .toggle, and .contains.

Queries

  • document object:
    • Event listeners: addEventListener for multiple event handlers.
    • Selecting elements:
      • .getElementById returns an object.
      • .getElementsByName returns a list.
      • .getElementsByTagName returns a list.
      • .getElementsByClassName returns a list.
      • .querySelector for powerful selection using CSS syntax (e.g., .class, #id, ..).
      • .querySelectorAll to select all matching elements.
    • Operations:
      • .Events property for events.
      • .value property for input values.

Events

  • List of events:
    • DOMContentLoaded, click, mouseOver, mouseOut, mouseDown, mouseUp, keydown, keyup, load, blur, change, checked.
  • Family Relations:
    • .firstElementChild, .lastElementChild, .parentElement, .nextElementSibling, .previousElementSibling.
    • .children selects one object and returns an object.
    • Array.from(.children) returns an array.

Window

  • Used to interact with the web browser.
  • this refers to the window object.
  • console object for console operations:
    • .log, .prompt, .dir, .alert.
  • window object:
    • Properties: innerWidth, innerHeight, scrollX, scrollY, location.
      • .href shows the URL and can be used to navigate to another page.
      • .hostname, .pathname properties.
    • Methods: .open (opens a new window), .close, .print, .confirm, .alert, .prompt.

Variables

  • let, const, and var: Keywords used for variable declaration and assignment.
    • let: Declares a block-scoped variable that can be reassigned.
    • const: Declares a block-scoped variable that cannot be reassigned.
    • var: Declares a function-scoped variable (older way of declaring variables, generally not recommended).
  • Data types: JavaScript has several built-in data types.
    • String: Represents textual data and is enclosed in single or double quotes.
    • Number: Represents numeric values, including integers and floating-point numbers.
    • Boolean: Represents either true or false.
    • Object: Represents a collection of key-value pairs.
    • Array: Represents an ordered list of values.
    • null: Represents the absence of any object value.
    • undefined: Represents an uninitialized variable or missing property.

Functions

  • Function declaration and invocation:
    • Declaring a function: function functionName(parameters) { // code }
    • Invoking a function: functionName(arguments)
  • Parameters and arguments:
    • Parameters: Variables listed in the function declaration that represent the values passed to the function.
    • Arguments: Values passed to a function when it is invoked.
  • Return statement:
    • Used to specify the value that the function should return when it is called.
    • The return statement ends the function execution and sends the specified value back to the caller.

Control Flow

  • Conditional statements:
    • if: Executes a block of code if a specified condition is true.
    • else if: Allows for additional conditions to be checked if the initial if condition is false.
    • else: Specifies a block of code to be executed if none of the preceding conditions are true.
  • Comparison operators: Used to compare values.
    • === (strict equality): Checks if two values are equal in value and type.
    • !== (strict inequality): Checks if two values are not equal in value or type.
    • <, >, <=, >=: Less than, greater than, less than or equal to, greater than or equal to.
  • Logical operators: Usedto combine or negate conditions.
    • && (logical AND): Returns true if both conditions are true.
    • || (logical OR): Returns true if at least one of the conditions is true.
    • ! (logical NOT): Negates a condition, returning true if the condition is false, and vice versa.
  • Loops:
    • for: Executes a block of code a specified number of times.
    • while: Executes a block of code as long as a specified condition is true.
    • do...while: Executes a block of code once, and then repeats the execution as long as a specified condition is true.
    • break: Terminates the execution of a loop.
    • continue: Skips the current iteration of a loop and continues with the next iteration.
  • Switch statement:
    • Evaluates an expression and executes code associated with a matching case label.
    • The break statement is used to exit the switch statement.
    • If no matching case label is found, the code under the default label is executed (optional).

Error Handling

  • Try-Catch statement:
    • try: Contains the code that might throw an exception.
    • catch: Specifies the code to be executed if an exception is thrown in the try block.
    • finally: Specifies a block of code to be executed regardless of whether an exception is thrown or caught.
    • throw: Used to throw a custom exception.

Arrays

  • Creating an array:
    • let arrayName = [] or let arrayName = new Array().
  • Accessing array elements: arrayName[index].
  • Modifying array elements: arrayName[index] = value.
  • Array methods:
    • .push(): Adds one or more elements to the end of an array and returns the new length of the array.
    • .pop(): Removes the last element from an array and returns that element.
    • .shift(): Removes the first element from an array and returns that element.
    • .unshift(): Adds one or more elements to the beginning of an array and returns the new length of the array.
    • .slice(): Returns a shallow copy of a portion of an array into a new array object.
    • .splice(): Changes the contents of an array by removing, replacing, or adding elements.
    • .concat(): Combines two or more arrays and returns a new array.
    • .join(): Joins all elements of an array into a string.
    • .indexOf(): Returns the first index at which a given element can be found in an array, or -1 if it is not present.
    • .lastIndexOf(): Returns the last index at which a given element can be found in an array, or -1 if it is not present.
    • .includes(): Determines whether an array contains a specific element.
    • .forEach(): Executes a provided function once for each array element.
    • .map(): Creates a new array with the results of calling a provided function on every element in the array.
    • .filter(): Creates a new array with all elements that pass the test implemented by the provided function.
    • .reduce(): Applies a function against an accumulator and each element in the array to reduce it to a single value.
    • .sort(): Sorts the elements of an array in place and returns the sorted array.
    • .reverse(): Reverses the order of the elements in an array in place.
    • .isArray(): Determines whether the passed value is an array.
    • .length: Property that returns the number of elements in an array.
  • Array destructuring: Assigns values from an array to variables in a single statement.

Objects

  • Creating an object:
    • let objectName = {} or let objectName = new Object().
    • Can also be created using object constructors and classes.
  • Accessing object properties:
    • Dot notation: objectName.propertyName.
    • Bracket notation: objectName["propertyName"].
  • Modifying object properties: objectName.propertyName = value.
  • Object methods:
    • .hasOwnProperty(): Returns a boolean indicating whether the object has the specified property as a direct property of itself.
    • .keys(): Returns an array containing the names of all of the object's own enumerable properties.

    • .entries(): Returns an array containing all of the object's own enumerable property [key, value] pairs.
